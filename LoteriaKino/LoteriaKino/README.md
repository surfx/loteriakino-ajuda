# Install

```
dotnet add package Selenium.WebDriver -v 4.17.0 -s https://api.nuget.org/v3/index.json
dotnet add package WebDriverManager -v 2.17.1 -s https://api.nuget.org/v3/index.json
```

## Chrome Driver download

Baixar em [Chrome driver](https://chromedriver.chromium.org/downloads), descompactar `chromedriver.exe` para:

`C:\Program Files (x86)\webdriver\bin`

Adicionar o PATH: `C:\Program Files (x86)\webdriver\bin`

No código: `IWebDriver driver = new ChromeDriver(@"C:\Program Files (x86)\webdriver\bin", options);`
